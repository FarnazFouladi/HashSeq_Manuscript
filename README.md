<!-- badges: start -->

This repository includes scripts for analyses of the manuscript 
"HashSeq: A Simple, Scalable and Conservative Denovo Variant Caller for 16S rRNA Gene Datasets". In order to generate the figures in the s manuscript, set your work directory in the "HashMap_Manuscript" directory. R scripts in codes/figures as well as codes/blast/blast.R
generate the figures found in the "figures" directory.
